# Change Log Entry Template

Use this template when recording plan changes during `/calibrate`.

**Location:** `.beads/change-log.md`

---

## Template

```markdown
### <YYYY-MM-DD> — <Scope>

**Trigger:** Scheduled / Ad-Hoc / User Request
**Owner:** <agent-name>
**Context:** <why this change is happening>

**Decision Summary:**
- Keep: ...
- Change: ...
- Cut: ...
- Add: ...

**Impacted Phases/Beads:**
- Phase X → [what changed]
- Bead Y → [what changed]

**Rationale:** <why this is better aligned with the North Star>
**Risks Introduced:** <new risks or tradeoffs>
**Assumptions Invalidated:** <what changed>
**User Clarifiers Needed:** <questions to ask>
**Evidence:** <links to calibration summary, tests, files>
```
